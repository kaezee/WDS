Cards

Container components for marketplace listings, content, and information display.

Usage & Purpose

Use cards when: Displaying products in marketplace grids, showing supplier profiles, organizing content like blog posts, or presenting structured information in dashboards.

Don't use cards for: Single pieces of information that don't need visual separation, or when a simple list would be clearer.

Product Cards

Main marketplace listing cards with 4:3 images, no public pricing, and request quote CTAs. Display product details, supplier info, certifications, and attributes.

JSX 
const ProductCard = ({ 
  product, 
  label, 
  certifications = [], 
  attributes = [] 
}) => {
  return (
    <div className="card-product">
      <div className="card-product-image">
        {label && (
          <span className={`status-overlay status-overlay-positioned status-overlay-${label.type}`}>
            {label.text}
          </span>
        )}
        <img src={product.image} alt={product.name} />
      </div>
      <div className="card-product-content">
        {/* Positive Attributes ABOVE title */}
        {attributes.length > 0 && (
          <div className="card-product-attributes">
            {attributes.map(attr => (
              <span key={attr.text} className={`status-text status-text-${attr.type}`}>
                <span className="status-text-icon">{attr.icon}</span>
                {attr.text}
              </span>
            ))}
          </div>
        )}
        
        <h3 className="card-product-title">
          {product.name} Min Order {product.minOrder} kg
        </h3>
        <a href={`/supplier/${product.supplierId}`} className="card-product-supplier">
          {product.supplier}
        </a>
        
        {certifications.length > 0 && (
          <div className="card-product-certifications">
            {certifications.map(cert => (
              <span key={cert.name} className={`cert-badge cert-badge-${cert.color}`}>
                <span className="cert-badge-icon"></span>
                {cert.name}
              </span>
            ))}
          </div>
        )}
        
        <button className="btn btn-md btn-primary card-product-cta">
          Request Quote
        </button>
      </div>
    </div>
  );
};

// Usage
<ProductCard 
  product={{
    name: "Atlantic Salmon Fillets",
    supplier: "Norwegian Seafoods Co.",
    supplierId: "norwegian-seafoods",
    image: "/images/salmon.jpg",
    minOrder: 50
  }}
  label={{ type: 'new', text: 'New' }}
  certifications={[
    { name: 'MSC', color: 'blue' },
    { name: 'ASC', color: 'blue' }
  ]}
  attributes={[
    { type: 'popular', text: 'Popular', icon: '⭐' }
  ]}
/>
