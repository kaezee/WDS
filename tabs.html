Tabs

Organize content into separate views with tabbed navigation.

Usage & Purpose

Use tabs when: Organizing related content into separate sections, displaying product details with different categories, or showing settings panels.

Don't use tabs for: Sequential processes (use stepper), navigation between pages (use breadcrumbs), or more than 7-8 tabs (use dropdown or sidebar).

Default Tabs

Standard underline tabs for content organization.

Product overview content goes here. This tab shows general information about the product.

Technical specifications and product details are displayed in this tab.

Certification information including MSC, ASC, and other quality standards.

Customer reviews and ratings are shown here.

HTML + CSS 
<div class="tabs" id="my-tabs">
  <div class="tabs-list" role="tablist">
    <button class="tab active" onclick="switchTab('my-tabs', 0)">
      Overview
    </button>
    <button class="tab" onclick="switchTab('my-tabs', 1)">
      Specifications
    </button>
    <button class="tab" onclick="switchTab('my-tabs', 2)">
      Certifications
    </button>
    <button class="tab" onclick="switchTab('my-tabs', 3)">
      Reviews
    </button>
  </div>
  
  <div class="tab-panels">
    <div class="tab-panel active">
      <!-- Overview content -->
    </div>
    <div class="tab-panel">
      <!-- Specifications content -->
    </div>
    <div class="tab-panel">
      <!-- Certifications content -->
    </div>
    <div class="tab-panel">
      <!-- Reviews content -->
    </div>
  </div>
</div>

<script>
function switchTab(containerId, index) {
  const container = document.getElementById(containerId);
  const tabs = container.querySelectorAll('.tab');
  const panels = container.querySelectorAll('.tab-panel');
  
  tabs.forEach((tab, i) => {
    tab.classList.toggle('active', i === index);
  });
  
  panels.forEach((panel, i) => {
    panel.classList.toggle('active', i === index);
  });
}
</script>
JSX 
const Tabs = ({ tabs, children }) => {
  const [activeTab, setActiveTab] = useState(0);

  return (
    <div className="tabs">
      <div className="tabs-list" role="tablist">
        {tabs.map((tab, index) => (
          <button
            key={index}
            className={`tab ${activeTab === index ? 'active' : ''}`}
            onClick={() => setActiveTab(index)}
          >
            {tab}
          </button>
        ))}
      </div>
      
      <div className="tab-panels">
        {React.Children.map(children, (child, index) => (
          <div className={`tab-panel ${activeTab === index ? 'active' : ''}`}>
            {child}
          </div>
        ))}
      </div>
    </div>
  );
};

// Usage
<Tabs tabs={['Overview', 'Specifications', 'Certifications', 'Reviews']}>
  <div>Overview content</div>
  <div>Specifications content</div>
  <div>Certifications content</div>
  <div>Reviews content</div>
</Tabs>
