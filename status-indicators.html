Status Indicators

Dynamic status displays for orders and inventory. Three styles: outlined order tracking, urgent overlays, and positive attribute badges.

Three Types of Status Indicators

Order Status (Outlined)

Order tracking states with colored borders. Use on order pages, dashboards, and order history. Examples: Delivered, Processing, Cancelled.

Urgent Overlays

High-contrast overlays on product cards. Small, bold, attention-grabbing. Examples: Limited Stock, New Arrival, Sale.

Positive Badges

Inline badges near product titles. Colored backgrounds with icons. Examples: Fresh Catch, Bestseller, Trending.

Not Certification Badges: Status indicators are different from certification badges (Sustainable, MSC). They show temporary states, not permanent credentials.

Order Status Indicators

Outlined pills for order tracking. Use for Delivered, Processing, Cancelled, and other order states.

Delivered Processing Cancelled Archived Pending Shipped 
<span class="order-status order-status-delivered">Delivered</span>
<span class="order-status order-status-processing">Processing</span>
<span class="order-status order-status-cancelled">Cancelled</span>
<span class="order-status order-status-archived">Archived</span>
const OrderStatus = ({ status }) => {
  const statusClass = `order-status order-status-${status.toLowerCase()}`;
  return <span className={statusClass}>{status}</span>;
};

// Usage
<OrderStatus status="Delivered" />
<OrderStatus status="Processing" />
<OrderStatus status="Cancelled" />

const ProductCard = ({ title, price, status, image }) => (
  <div className="product-card" style={{ position: 'relative' }}>
    <div className="product-image">
      {status && (
        <span className={`status-overlay status-overlay-${status} status-overlay-positioned`}>
          {status.toUpperCase()}
        </span>
      )}
      <img src={image} alt={title} />
    </div>
    <h3>{title}</h3>
    <p>{price}</p>
  </div>
);

// Usage
<ProductCard title="Sea Bass" price="$16.80/kg" status="limited" />

Positive Attribute Badges

Inline badges with colored backgrounds and optional icons. Use near product titles for quality indicators.

Fresh Catch Bestseller Trending Popular 
<span class="status-badge status-badge-fresh">
  <span class="status-badge-icon"></span>
  Fresh Catch
</span>

<!-- With actual icon -->
<span class="status-badge status-badge-bestseller">
  <span class="status-badge-icon">
    <img src="/icons/star.svg" alt="" />
  </span>
  Bestseller
</span>
const StatusBadge = ({ type, children, icon }) => (
  <span className={`status-badge status-badge-${type}`}>
    {icon ? (
      <span className="status-badge-icon">
        <img src={icon} alt="" />
      </span>
    ) : (
      <span className="status-badge-icon"></span>
    )}
    {children}
  </span>
);

// Usage
<StatusBadge type="fresh">Fresh Catch</StatusBadge>
<StatusBadge type="bestseller" icon="/icons/star.svg">Bestseller</StatusBadge>
