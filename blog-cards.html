Blog Cards

Content marketing cards with featured images, category tags, and article metadata.

Usage & Purpose

Use blog cards when: Displaying blog posts, articles, news items, case studies, or any content marketing materials in grid layouts.

Don't use blog cards for: Product listings (use Product Cards), supplier profiles (use Supplier Cards), or non-content pages.

Blog Card Variants

Cards with 16:9 featured images, category overlays, titles, excerpts, and metadata. Hover changes background color.

JSX 
const BlogCard = ({ post }) => {
  return (
    <div className="card-blog">
      <div className="card-blog-image">
        <span className="card-blog-category">{post.category}</span>
        <img src={post.image} alt={post.title} />
      </div>
      <div className="card-blog-content">
        <h3 className="card-blog-title">{post.title}</h3>
        <p className="card-blog-excerpt">{post.excerpt}</p>
        <div className="card-blog-meta">
          <span>{post.readTime} min read</span>
          <span className="card-blog-meta-divider"></span>
          <span>{post.date}</span>
        </div>
        <a href={post.url} className="card-blog-link">Read more</a>
      </div>
    </div>
  );
};

// Usage
<BlogCard 
  post={{
    title: "The Future of Sustainable Seafood Sourcing",
    category: "Sustainability",
    excerpt: "Discover how modern aquaculture practices are revolutionizing...",
    image: "/images/blog/sustainable-seafood.jpg",
    readTime: 5,
    date: "Jan 28, 2026",
    url: "/blog/sustainable-seafood"
  }}
/>

// Grid Layout
<div className="card-grid-blog">
  {posts.map(post => (
    <BlogCard key={post.id} post={post} />
  ))}
</div>
