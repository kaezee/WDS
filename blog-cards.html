Blog Cards

Article preview cards with featured images, category tags, and reading metadata for content marketing and industry insights.

Usage & Purpose

Blog Cards display article previews in content sections, resource hubs, and related content modules. They feature 16:9 images for visual impact, category overlays for quick classification, and reading time metadata to set user expectations. Best used in 3-column grids for blog listings or 2-column layouts for featured content sections.

Blog Cards - 3 Column Grid

Display multiple article previews in a responsive grid layout with category tags, titles, excerpts, and metadata.

const BlogCard = ({ category, image, title, excerpt, readTime, date, href }) => (
  <div className="card-blog">
    <div className="card-blog-image">

const BlogGrid = ({ articles }) => (
  <div className="card-grid-blog">
