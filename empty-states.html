Empty States

Guide users when no content is available and provide clear next actions.

Usage & Purpose

Use empty states when: No data exists yet, search returns no results, filters remove all items, or an error prevents content display.

Don't use empty states for: Loading states (use skeleton or spinner), temporary network issues (use error message), or when content is intentionally hidden.

No Orders

Empty state when user has no orders yet.

No orders yet

You haven't placed any orders. Start browsing our catalog to find quality seafood products.

HTML + CSS 
<div class="empty-state">
  <div class="empty-state-icon">
    <span class="empty-state-icon-inner"></span>
  </div>
  <h3 class="empty-state-title">No orders yet</h3>
  <p class="empty-state-description">
    You haven't placed any orders. Start browsing our catalog 
    to find quality seafood products.
  </p>
  <div class="empty-state-actions">
    <button class="btn btn-md btn-primary">Browse Products</button>
  </div>
</div>
JSX 
const EmptyState = ({ title, description, action, onAction }) => {
  return (
    <div className="empty-state">
      <div className="empty-state-icon">
        <span className="empty-state-icon-inner"></span>
      </div>
      <h3 className="empty-state-title">{title}</h3>
      <p className="empty-state-description">{description}</p>
      <div className="empty-state-actions">
        <button className="btn btn-md btn-primary" onClick={onAction}>
          {action}
        </button>
      </div>
    </div>
  );
};

// Usage
<EmptyState
  title="No orders yet"
  description="You haven't placed any orders. Start browsing our catalog to find quality seafood products."
  action="Browse Products"
  onAction={() => navigate('/products')}
/>

No Search Results

Empty state when search returns no results.

No results found

We couldn't find any products matching "lobster tails". Try adjusting your search or browse our categories.

HTML + CSS 
<div class="empty-state empty-state-search">
  <div class="empty-state-icon">
    <span class="empty-state-icon-inner"></span>
  </div>
  <h3 class="empty-state-title">No results found</h3>
  <p class="empty-state-description">
    We couldn't find any products matching "lobster tails". 
    Try adjusting your search or browse our categories.
  </p>
  <div class="empty-state-actions">
    <button class="btn btn-md btn-primary-outline">Clear Search</button>

  </div>
</div>
JSX 
<EmptyState
  variant="search"
  title="No results found"
  description={`We couldn't find any products matching "${query}". Try adjusting your search or browse our categories.`}
  actions={[
    { label: 'Clear Search', onClick: handleClear, variant: 'primary-outline' },
    { label: 'Browse Categories', onClick: handleBrowse, variant: 'secondary-outline' }
  ]}
/>
