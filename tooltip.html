Tooltip

Provide contextual help and additional information on hover.

Usage & Purpose

Use tooltips when: Providing brief explanations for icons, abbreviations, or truncated text, showing additional context without cluttering the interface.

Don't use tooltips for: Critical information users must see, long explanations (use modal), interactive content, or mobile-primary interfaces (tooltips don't work well on touch).

Basic Tooltip

Default dark tooltip on hover.

Hover over MSC Certified Marine Stewardship Council for more info

HTML + CSS 
<span class="tooltip tooltip-top">
  <span>MSC Certified</span>
  <span class="tooltip-content">
    Marine Stewardship Council
    <span class="tooltip-arrow"></span>
  </span>
</span>
JSX 
const Tooltip = ({ children, content, position = 'top' }) => {
  return (
    <span className={`tooltip tooltip-${position}`}>
      {children}
      <span className="tooltip-content">
        {content}
        <span className="tooltip-arrow"></span>
      </span>
    </span>
  );
};

// Usage
<Tooltip content="Marine Stewardship Council" position="top">
  <span>MSC Certified</span>
</Tooltip>

Positions

Top, right, bottom, and left placement.

Tooltip on top Tooltip on right Tooltip on bottom Tooltip on left

<!-- Top -->
<span class="tooltip tooltip-top">...</span>

<!-- Right -->
<span class="tooltip tooltip-right">...</span>

<!-- Bottom -->
<span class="tooltip tooltip-bottom">...</span>

<!-- Left -->
<span class="tooltip tooltip-left">...</span>
JSX 
<Tooltip content="Tooltip text" position="top">...</Tooltip>
<Tooltip content="Tooltip text" position="right">...</Tooltip>
<Tooltip content="Tooltip text" position="bottom">...</Tooltip>
<Tooltip content="Tooltip text" position="left">...</Tooltip>

  <span class="tooltip-content">
    Country where the seafood was caught or farmed

  </span>
</span>

<Tooltip content="Country where the seafood was caught or farmed">
  <span className="tooltip-icon">?</span>
</Tooltip>

Multiline Tooltip

Tooltip with longer content that wraps to multiple lines.

This tooltip contains multiple lines of text to provide more detailed information when needed.
HTML + CSS 
<span class="tooltip tooltip-multiline tooltip-top">
  <button>Hover for details</button>
  <span class="tooltip-content">
    This tooltip contains multiple lines of text to provide 
    more detailed information when needed.
    <span class="tooltip-arrow"></span>
  </span>
</span>
JSX 
<Tooltip 
  multiline 
  content="This tooltip contains multiple lines of text to provide more detailed information when needed."
>
  <button>Hover for details</button>
</Tooltip>
