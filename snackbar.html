Snackbar / Toast

Temporary notification messages that provide feedback on user actions without interrupting workflow.

Usage & Purpose

Use snackbars when: You need to provide brief, non-critical feedback that auto-dismisses (order confirmations, save notifications, background process updates).

Don't use snackbars for: Critical errors requiring immediate action (use modals instead), lengthy messages, or information users need to reference later.

Snackbar Variants

Five variants for different message types, using WhaleLord's darker, more sophisticated semantic colors.

JSX 
const Snackbar = ({ variant = 'default', message, onClose, icon }) => {
  return (
    <div className={`snackbar snackbar-${variant}`}>
      {icon && (
        <svg className="snackbar-icon" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          {icon}
        </svg>
      )}
      <span>{message}</span>
      <button className="snackbar-close" onClick={onClose}>✕</button>
    </div>
  );
};

JSX 
const Snackbar = ({ 
  variant = 'default', 
  message, 
  onClose, 
  icon,
  actionLabel,
  onAction 
}) => {
  return (
    <div className={`snackbar snackbar-${variant}`}>
      {icon && (
        <svg className="snackbar-icon" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          {icon}
        </svg>
      )}
      <span>{message}</span>
      {actionLabel && (
        <button className="snackbar-action" onClick={onAction}>
          {actionLabel}
        </button>
      )}
      <button className="snackbar-close" onClick={onClose}>✕</button>
    </div>
  );
};

// Usage
<Snackbar 
  variant="success" 
  message="Item added to cart"
  icon={<path strokeLinecap="round" strokeLinejoin="round" strokeWidth="2" d="M5 13l4 4L19 7" />}
  actionLabel="View Cart"
  onAction={() => navigate('/cart')}
  onClose={() => console.log('Dismissed')}
/>

JSX 
const Snackbar = ({ 
  variant = 'default', 
  position = 'bottom-center', // 'bottom-center' | 'top-right' | 'top-center'
  message, 
  onClose, 
  icon 
}) => {
  const positionClass = position !== 'bottom-center' ? `snackbar-${position}` : '';
  
  return (
    <div className={`snackbar snackbar-${variant} ${positionClass}`.trim()}>
      {icon && (
        <svg className="snackbar-icon" fill="none" stroke="currentColor" viewBox="0 0 24 24">
          {icon}
        </svg>
      )}
      <span>{message}</span>
      <button className="snackbar-close" onClick={onClose}>✕</button>
    </div>
  );
};

// Usage
<Snackbar 
  variant="info" 
  position="top-right"
  message="New notification"
  onClose={() => console.log('Dismissed')}
/>
