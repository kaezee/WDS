Checkout Cards

Order summary cards for checkout flows, dashboards, and order history.

Usage & Purpose

Use checkout cards when: Displaying order summaries in checkout, showing order history, or presenting order details in dashboards.

Don't use checkout cards for: Product listings (use Product Cards), cart items (use list layout), or detailed invoices (use full page).

Order Summary Card

Display order details with line items, totals, status, and action buttons.

JSX 
const OrderCard = ({ order }) => {
  return (
    <div className="card-order">
      <div className="card-order-header">
        <div>
          <h3 className="card-order-number">Order #{order.number}</h3>
          <p className="card-order-date">{order.date}</p>
        </div>
        <span className={`order-status order-status-${order.status.toLowerCase()}`}>
          {order.status}
        </span>
      </div>
      
      <div className="card-order-items">
        {order.items.map((item, index) => (
          <div key={index} className="card-order-item">
            <span className="card-order-item-name">{item.name}</span>
            <span className="card-order-item-quantity">{item.quantity} kg</span>
          </div>
        ))}
      </div>
      
      <div className="card-order-total">
        <span className="card-order-total-label">Total Weight</span>
        <span className="card-order-total-amount">{order.totalWeight} kg</span>
      </div>
      
      <div className="card-order-actions">
        <button className="btn btn-md btn-primary" style={{ flex: 1 }}>
          {order.primaryAction}
        </button>
        <button className="btn btn-md btn-secondary" style={{ flex: 1 }}>
          {order.secondaryAction}
        </button>
      </div>
    </div>
  );
};

// Usage
<OrderCard 
  order={{
    number: "WHD-2846",
    date: "January 28, 2026",
    status: "Processing",
    items: [
      { name: "Atlantic Salmon Fillets", quantity: 150 },
      { name: "Norwegian Cod Premium", quantity: 200 },
      { name: "Icelandic Halibut", quantity: 75 }
    ],
    totalWeight: 425,
    primaryAction: "View Details",
    secondaryAction: "Download Invoice"
  }}
/>
