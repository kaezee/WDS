Certification Badges

Trust signals for quality certifications, compliance standards, and product attributes. Compact, professional badges using DM Sans with 6px radius.

What Are Certification Badges?

Certification badges are visual indicators of quality seals, compliance standards, and trusted certifications. They build buyer confidence by displaying credentials like MSC Certified, Organic, Sustainable, or ISO standards. These badges are NOT for status indicators like "New" or "Limited Stock" - those are separate components.

When to Use Certification Badges

  • ✅ Environmental certifications (Sustainable, Organic, Eco-Friendly)
  • ✅ Marine/seafood certifications (MSC, ASC, Friend of the Sea)
  • ✅ Compliance standards (ISO 22000, HACCP, FDA Approved)
  • ✅ Quality seals and trust signals

When NOT to Use

  • ❌ Status indicators ("New", "Limited Stock", "Fresh") - use status indicators instead
  • ❌ Promotional tags ("Sale", "Deal", "Hot") - use status indicators
  • ❌ Action buttons - use actual buttons

Green Badges (Sustainable/Organic)

Use green badges for environmental certifications: Sustainable, Organic, Eco-Friendly, Grass-Fed.

Sustainable Organic Eco-Friendly
HTML 
<span class="cert-badge cert-badge-green">
  <span class="cert-badge-icon"></span>
  Sustainable
</span>

<!-- With actual icon -->
<span class="cert-badge cert-badge-green">
  <span class="cert-badge-icon">
    <img src="/logos/organic.svg" alt="Organic" />
  </span>
  Organic
</span>
React + JSX 
const GreenBadge = ({ children, icon }) => (
  <span className="cert-badge cert-badge-green">
    {icon ? (
      <span className="cert-badge-icon">
        <img src={icon} alt="" />
      </span>
    ) : (
      <span className="cert-badge-icon"></span>
    )}
    {children}
  </span>
);

// Usage
<GreenBadge>Sustainable</GreenBadge>
<GreenBadge icon="/logos/organic.svg">Organic</GreenBadge>

Best Practices

✅ Do

  • Use green for environmental certifications
  • Use blue for marine/seafood certifications
  • Use gray for standard compliance
  • Place near product title or image
  • Include certification logo when available
  • Keep text concise (2-3 words max)
  • Use actual certification names

❌ Don't

  • Don't use for status indicators ("New", "Limited")
  • Don't use for promotional tags ("Sale", "Hot Deal")
  • Don't make badges clickable (not buttons)
  • Don't use multiple color variants on one badge
  • Don't use badges without certification backing
  • Don't make text too long (breaks layout)
  • Don't stack more than 3 badges per product

♿ Accessibility Guidelines

✓

Color contrast: All variants meet WCAG AA minimum (4.5:1), blue/gray exceed AAA

✓

Font: DM Sans at 12px with 600 weight ensures readability

✓

Icons: Include alt text when using actual certification logos

✓

Not interactive: Badges are visual indicators only, not focusable

✓

Screen readers: Text content is naturally accessible
