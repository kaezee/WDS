Breadcrumbs

Show navigation hierarchy and help users understand their current location.

Usage & Purpose

Use breadcrumbs when: Navigation has multiple levels (3+), users need to backtrack easily, or showing the current page's context within the site hierarchy.

Don't use breadcrumbs for: Single-level navigation, linear processes (use stepper), or when the path is obvious from the page content.

Standard breadcrumb navigation with chevron separators.

HTML + CSS 
<nav class="breadcrumbs" aria-label="Breadcrumb">
  <div class="breadcrumb-item">
    <a href="/" class="breadcrumb-link">Home</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <div class="breadcrumb-item">
    <a href="/products" class="breadcrumb-link">Products</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <div class="breadcrumb-item">
    <a href="/products/seafood" class="breadcrumb-link">Seafood</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <div class="breadcrumb-item">
    <span class="breadcrumb-current">Atlantic Salmon</span>
  </div>
</nav>
JSX 
const Breadcrumbs = ({ items }) => {
  return (
    <nav className="breadcrumbs" aria-label="Breadcrumb">
      {items.map((item, index) => (
        <React.Fragment key={index}>
          {index > 0 && <span className="breadcrumb-separator"></span>}
          <div className="breadcrumb-item">
            {index === items.length - 1 ? (
              <span className="breadcrumb-current">{item.label}</span>
            ) : (
              <a href={item.href} className="breadcrumb-link">
                {item.label}
              </a>
            )}
          </div>
        </React.Fragment>
      ))}
    </nav>
  );
};

// Usage
<Breadcrumbs 
  items={[
    { label: 'Home', href: '/' },
    { label: 'Products', href: '/products' },
    { label: 'Seafood', href: '/products/seafood' },
    { label: 'Atlantic Salmon', href: '/products/seafood/salmon' }
  ]} 
/>

Collapsed (Long Paths)

Breadcrumbs with ellipsis for long navigation paths.

Home
Products
Seafood
Fresh Fish
...
Salmon
Atlantic Salmon Fillet
HTML + CSS 
<nav class="breadcrumbs breadcrumbs-collapsed" aria-label="Breadcrumb">
  <div class="breadcrumb-item">
    <a href="/" class="breadcrumb-link">Home</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <!-- Middle items (hidden by CSS) -->
  <div class="breadcrumb-item">
    <a href="/products" class="breadcrumb-link">Products</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <div class="breadcrumb-ellipsis">
    <span>...</span>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <!-- Last two items always shown -->
  <div class="breadcrumb-item">
    <a href="/salmon" class="breadcrumb-link">Salmon</a>
  </div>
  <span class="breadcrumb-separator"></span>
  
  <div class="breadcrumb-item">
    <span class="breadcrumb-current">Atlantic Salmon Fillet</span>
  </div>
</nav>
JSX 
const CollapsedBreadcrumbs = ({ items }) => {
  const showEllipsis = items.length > 4;
  
  return (
    <nav className="breadcrumbs" aria-label="Breadcrumb">
      {/* First item */}
      <div className="breadcrumb-item">
        <a href={items[0].href} className="breadcrumb-link">
          {items[0].label}
        </a>
      </div>
      <span className="breadcrumb-separator"></span>
      
      {showEllipsis && (
        <>
          <div className="breadcrumb-ellipsis">...</div>
          <span className="breadcrumb-separator"></span>
        </>
      )}
      
      {/* Last two items */}
      {items.slice(-2).map((item, index) => (
        <React.Fragment key={index}>
          <div className="breadcrumb-item">
            {index === 1 ? (
              <span className="breadcrumb-current">{item.label}</span>
            ) : (
              <a href={item.href} className="breadcrumb-link">{item.label}</a>
            )}
          </div>
          {index === 0 && <span className="breadcrumb-separator"></span>}
        </React.Fragment>
      ))}
    </nav>
  );
};
